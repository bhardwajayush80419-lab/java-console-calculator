# Java Console Calculator

A console-based calculator built in Java as part of the Elevate Labs Java Developer Internship — Task 1.

## Features
- Addition, Subtraction, Multiplication, Division
- Loops continuously until user chooses to exit
- Handles divide-by-zero gracefully
- Input validation (handles non-numeric input without crashing)
- Clean, readable console output

## Tools Used
- Java (JDK 8+)
- IntelliJ IDEA CE
- Terminal / Command Prompt

## How to Run

1. Make sure Java is installed. Check with:
   javac -version

2. Compile the file:
   javac Calculator.java

3. Run the program:
   java Calculator
